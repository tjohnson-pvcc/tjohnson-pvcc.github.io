    • Term Definition
    LAN A computer network that normally consists of a collection of computers in a single building or building complex
    MAN a network of intermediate size, such as one spanning a local community
    WAN Network that links machines over a greater distance—perhaps in neighboring cities or on opposite sides of the world.
    Bus network popular network topology that communicates directly with each other over a common bus
    Star netowrk popular network topology that communicates indirectly through an intermediary central machine.
    CSMA/CD Ethernet A protocol in a bus network that transmits messages
    CSMA/CA Wifi wireless networks that adopt the policy of trying to avoid collisions rather than trying to detect them.
    Repeater/td> a device that passes signals back and forth between the two original buses without considering the meaning of the signals
    Bridge looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
    Switch essentially a bridge with multiple connections, allowing it to connect several buses rather than just two.
    Router Devices that handles the connection between networks to form an internet
    Gateway The “point” at which one network is linked to an internet Client/server model
    P2P: peer-to-peer Server model whereas the client/server model involves one process (the server) providing a service to numerous others (clients), the peer-to-peer model involves processes that provide service to and receive service from each other
    Distributed system Software system that consists of software units that execute as processes on different computers
    Cloud computing a distributed system in which many independent computers work closely together to provide computation or services comparable to a much larger machine.
    Tier 1 ISP that consists of very high-speed, high-capacity, international WANs. These networks are thought of as the backbone of the Internet.
    Tier 2 ISPs that tend to be more regional in scope and less potent in their capabilities.
    Tier 3 Intermediary that provides access to the core of the router IP address
    ICANN is a non-profit corporation established to coordinate the Internet’s operation. (https://www.icann.org/)
    DNS Name servers that is used as an Internet-wide directory system
    TLD: Top-level domain: The suffix used to reflect the domain’s classification (Ex. .edu)
    Country-code TLDs Two letter TLDs for each specific country (https://www.worldstandards.eu/other/tlds/)
    FTP Application for listing and copying files across the network
    HTTP The communications protocol used to connect to Web servers on the Internet or on a local network
    SMTP defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
    URL Unique address each document is given that contains the information needed by a browser to contact the proper server and request the desired document.
    Client-side activities activities that are performed by a client (such as a browser)
    Server-side activities activities that are performed by a server (such as a web server).
    Application layer layer consists of those software units such as clients and servers that use Internet communication to carry out their tasks
    Transport layer adds sequence numbers to the small segments it produces so that the segments can be reassembled at the message’s destination.
    Network layer Decides in which direction a packet should be sent at each step along the packet’s path through the Internet.
    Link layer Deals with the communication details particular to the individual network in which the computer resides.
    Routing on the Internet is: An packet-switching network, fault tolerant, and redundant
    Malware Software that might be transferred to, and executed on, the computer itself, or it might attack the computer from a distance.
    Virus software that infects a computer by inserting itself into programs that already reside in the machine.
    Worm an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
    Trojan horse is a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.
    Spyware software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
    Phishing a technique of obtaining information explicitly by simply asking for it.
    DDos attack the process of overloading a computer with messages.
    Firewall A primary prevention technique that filters traffic passing through a point in the network
    Spam filter Is a firewall designed to block unwanted emails.
    Proxy server a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
    Antivirus software used to detect and remove the presence of known viruses and other infections.
    Cryptography the study of sending and receiving messages securely in the presence of adversaries.
    Public key encryption involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
