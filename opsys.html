Operating Systems Terminology

Term Definition
operating system the software that controls the overall operation of a computer
FIFO objects are removed from the queue in the order in which they arrived
interactive processing a program being executed to carry on a dialogue with the user through remote terminals
time-sharing operating systems that provide service to multiple users at the same time
multiprogramming time is divided into intervals and then the execution of each job is restricted to only one interval at a time
application software consists of the programs for performing tasks particular to the machine’s utilization
system software performs tasks that are common to computer systems in general
utility software consists of programs for performing activities that are fundamental to computer installations but not included in the operating system
graphical user interface (GUI) objects to be manipulated, such as files and programs, are represented pictorially on the display as icons
shell operating system that communicates with users through textual messages using a keyboard and monitor screen
kernel the internal part of an operating system
device driver the software units that communicate with the controllers (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine
file manager Software component whose job is to coordinate the use of the machine’s mass storage facilities
memory manager Software component which is charged with the task of coordinating the machine’s use of main memory
virtual memory large “fictional” memory space created by paging
boot strapping how the operating system gets started. It is this procedure that transfers the operating system from mass storage into main memory
ROM A program that is constructed from special nonvolatile memory cells. Its contents can be read but not altered.
deadlock the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
spooling technique of holding data for output at a later but more convenient time
super user This person gains highly privileged access to the operating system by identifying him- or herself as the administrator during the login procedure
multi-factor authentication When logging in, the user must provide a password, and also a special access code text that would be messaged to the user’s smartphone during the login process. A third set of authentication factors might depend on something related to who the user is.
